# Shopping-Cart-Web-App

---

## Description

It is basically a web application developed using **JavaScript (Express.js + React.js)** where we can buy digital products like **mobiles, smartwatches, headphones, laptops, tabs** by checking their product name, price, photo, and choosing quantity.  

We can see our purchased products in the cart, and we can update the quantity there as well, and finally, we can do a **checkout**.
